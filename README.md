Automation and analytics solutions for small businesses using chatbots, dashboards and workflow automation.

This repository contains **three practical demos** that show how businesses can automate customer communication and transform their data into useful insights.

---

# Demos

## Demo 1 — Professional WhatsApp Chatbot

Automated chatbot capable of answering common customer questions and capturing basic customer information.

Typical questions handled by the bot:

• prices
• opening hours
• reservations
• availability

Example interaction:

Customer:
"¿Cuál es el horario?"

Bot:
"Nuestro horario es de 9:00am a 7:00pm."

Customer:
"¿Cuánto cuesta?"

Bot:
"El precio del servicio es S/30."

Features:

• automated responses
• conversation flow control
• customer data capture
• simple conversation logic

Folder:

```
demo-1-chatbot_profesional
```

## Demo 2 — Sales Dashboard

Example of a simple analytics dashboard for small businesses.

The dashboard shows key business metrics such as:

• daily sales
• new customers
• number of inquiries
• sales conversion rate

This type of dashboard allows business owners to understand:

• how many customers ask for information
• how many become paying customers
• which channels generate more sales

Folder:

```
demo-2-sales_dashboard
```

---

## Demo 3 — Business Automation

Example of workflow automation for repetitive business tasks.

Automation flow:

Form → Google Sheet → Script → Report

Process:

Customer submits a form
↓
Data is stored in Google Sheets
↓
Automation script processes the information
↓
Business owner receives a summary report

This reduces manual work and improves organization.

Folder:

```
demo-3-automation
```
