# Demo 3 — Business Automation (Form → Report)

## Overview

This demo shows how repetitive business tasks can be automated.

Instead of manually collecting customer information and creating reports, the system automatically captures the data and generates summaries.

---

## Example Workflow

Customer fills a form
↓
Data is stored in Google Sheets
↓
Automation script processes the data
↓
Business owner receives a report

---

## Typical Use Cases

Small businesses often need to manage:

• reservations
• service requests
• customer inquiries
• order registrations

Automation reduces manual work and avoids losing information.

---

## Example Scenario

A customer requests a service using a form.

The system automatically:

1. Stores the request
2. Registers the customer
3. Updates the database
4. Generates a report for the owner

---

## Automation Flow

Form → Google Sheet → Script → Report
